# Быстрый старт

## 1. Установка зависимостей
```bash
pip3 install -r requirements.txt
```

## 2. Настройка OpenAI API (опционально)
Создайте файл `.env` в корне проекта:
```env
OPENAI_API_KEY=your_openai_api_key_here
```

Если API ключ не указан, система будет использовать мок-анализ для демонстрации.

## 3. Запуск сервера
```bash
python3 main.py
```

## 4. Открытие веб-интерфейса
Перейдите по адресу: http://localhost:8080

## 5. Тестирование API (опционально)
В другом терминале:
```bash
python3 test_api.py
```

## Возможности системы:
- ✅ Добавление резюме кандидатов
- ✅ Создание описаний вакансий
- ✅ Автоматический анализ релевантности
- ✅ Просмотр результатов анализа
- ✅ RESTful API для интеграции
